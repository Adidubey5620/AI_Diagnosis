# User Guide - AI Medical Diagnosis Assistant

## Introduction
The AI Medical Diagnosis Assistant is a powerful tool designed to assist healthcare professionals in analyzing medical images (X-rays, MRIs, CT scans) using advanced artificial intelligence.

## Features
- **AI Analysis**: Detects abnormalities and provides differential diagnoses.
- **Advanced Viewer**: interactive image viewer with zoom, pan, and annotation capabilities.
- **Reporting**: Generate professional PDF reports.
- **Multi-language**: Support for English, Spanish, and French.
- **Accessibility**: High contrast mode and screen reader support.

## Getting Started

### 1. Uploading an Image
- Navigate to the **Dashboard**.
- Drag and drop a medical image (DICOM, JPG, PNG) into the upload area or click to select a file.
- The system supports single file uploads up to 50MB.

### 2. Viewing Results
- Once analysis is complete, the dashboard will update.
- **Left Panel**: Interactive image viewer. Use mouse wheel to zoom, click and drag to pan.
- **Right Panel**: Diagnosis details, including findings, differential diagnosis, and recommendations.

### 3. Generating a Report
- Click the **"Generate Full Report"** button in the diagnosis panel.
- Fill in the patient details (Name, ID, Referring Physician).
- Click **"Generate PDF"** to download the report.

## Emergency Protocol
In case of critical findings, a red banner will appear with quick actions to contact emergency services.

## Troubleshooting
- **Upload Failed**: Ensure the file format is supported and under 50MB.
- **Analysis Error**: Check your internet connection. If the issue persists, contact support.
